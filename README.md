# 🎯 SSP Smart Stream Analyzer

A smart web-based tool that helps students choose the most suitable academic stream (PCM, PCB, Commerce, Arts) based on their **interests and abilities**.

This is not just a simple quiz — it uses a **weighted decision system**, **confidence analysis**, and **multi-factor evaluation** to provide personalized recommendations.

---

## 🚀 Features

### 🧠 Smart Decision Engine

* Uses **Interest + Ability sliders**
* Weighted scoring system for each stream
* Real-time calculation

### 📊 Advanced Analysis

* Normalized scoring (fair comparison)
* Weakness penalty system
* Core subject importance handling

### 🎯 Personalized Output

* Recommended stream with confidence %
* Alternative stream with gap analysis
* AI-like explanation (based on user inputs)

### 💼 Career Guidance

* Stream-based career suggestions
* Clustered career paths (not random lists)

### 🎨 UI/UX

* Clean SSP theme (dark maroon + white)
* Smooth animations
* Progress bar visualization
* Dark mode toggle

### 📸 Export Feature

* Download result as image
* High-resolution output (blur-free)

---

## 🧩 How It Works

1. User sets:

   * Interest (0–100)
   * Ability (0–100)

2. System calculates:

   * Combined score = (Interest × 0.6 + Ability × 0.4)

3. Each stream gets a score based on:

   * Trait weights
   * Core subject importance
   * Weakness penalties

4. Scores are normalized and ranked

5. Final output includes:

   * Top stream
   * Alternative stream
   * Confidence %
   * Explanation
   * Career paths
